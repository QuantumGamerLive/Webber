Alpha Flow

An Alpha Bot System options sweeps bot with data provided by BenderProfitBox and BlackBox.

Syntax

To display 10 most recent options sweeps use flow. You can futher filter down requests by including a ticker flow <ticker>.

Examples

  • The following command displays 10 most recent options sweeps.

    John Doe
    Aug 10, 2020

    flow

  • Alpha
    Aug 10, 2020

  • The following command displays 10 most recent options sweeps for AAPL.

    John Doe
    Aug 10, 2020

    flow aapl

  • Alpha
    Aug 10, 2020

  • The following command displays 10 most recent options sweeps for AMZN.

    John Doe
    Aug 10, 2020

    flow amzn

    Aug 10, 2020
